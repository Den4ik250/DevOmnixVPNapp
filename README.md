## What is DevOmnix VPN?

**DevOmnix VPN** is a fast, secure and easy-to-use VPN application for Android. Built on the powerful [Sing-box](https://github.com/SagerNet/sing-box) core with VLESS protocol, DevOmnix VPN provides reliable access to the free internet with a subscription-based service.

> No ads. No logs. Just a fast and stable connection.

## 🚀 Features

✅ **Simple interface** — one-tap connect/disconnect  
🌍 **Server selection** — choose from multiple servers with live ping  
⚡ **VLESS protocol** — modern, fast and censorship-resistant  
🔐 **Secure** — end-to-end encrypted tunnel  
📊 **Traffic stats** — real-time speed and usage display  
💳 **Subscription plans** — Starter, Standard, Premium, Group  
👛 **Wallet** — balance top-up and automatic payment deduction  
🎁 **Referral program** — earn bonuses for inviting friends  
📱 **Split tunneling** — choose which apps use VPN (Android)  
🌙 **Dark / Light mode**  
🔔 **Push notifications** — subscription reminders and updates

## ⚙️ How It Works

1. Open the app — your VPN config loads automatically
2. Tap the big button to connect
3. Choose a server with the **Select Server** button
4. Manage your subscription in the **Profile** tab

## 🛡️ Tech Stack

- **App**: Flutter 3.x (Android)
- **VPN Core**: [Sing-box](https://github.com/SagerNet/sing-box) with VLESS/Reality
- **Backend**: FastAPI + PostgreSQL + Redis
- **Panels**: [3X-UI](https://github.com/MHSanaei/3x-ui) for VLESS client management
- **Payments**: Lava.ru + CryptoBot
- **Notifications**: Telegram bot + FCM push
